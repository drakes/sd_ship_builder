What?

Quickly and easily design ships for the space combat miniatures game Silent Death.

This ship builder implements the Starcraft Design System found in Silent Death: The Next Millennium. You will need the rules to play.

Features:

  • Browser-based, no install required.
  • Design ships in all the standard classes: fighters, scouts, shuttles, freighters, and gunboats.
  • Arm them with all the standard cannons and warheads.
  • Add basic ship options.
  • Select crew skill levels (or optionally turn them off to see the cost of the ship without crew).
  • Damage tracks are generated on the fly with every change you make.

How?

Select a ship class. The rest of the design options should appear, and they speak for themselves (as long as you're familiar with the game).

If that doesn't work you probably don't have JavaScript enabled... or you're using a rather old browser, time to upgrade.

Tips

  • Arrow keys are your friends: click twice on a drop-down to give it focus, then cycle through options rapidly using your arrow keys.
  • Design another ship: click the header to start a fresh ship design in a new tab.
  • Track damage: click on any damage track box to mark damage in red. Click again on a damaged box to make repairs.
  • Offline use: save this page (and its associated .js and .css files), as a single file archive if your browser provides this option. Or go to GitHub, click Downloads to get the whole thing as a zip, extract all of the contents, and double-click index.html.
  • Go fullscreen: if you need more space pressing F11 in most browsers enters fullscreen mode. F11 again or ESC should exit fullscreen mode.
